# Mini Games for Slow Learners and Dyslexia

## 📌 Description
This project is a collection of interactive mini-games designed for slow learners and individuals with dyslexia. It helps improve reading, memory, and cognitive skills in a fun and engaging way.

## 🎮 Features
- Simple and user-friendly games
- Focus on learning through play
- Improves memory and attention
- Helps with reading and spelling skills

## 💻 Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download the project
2. Open index.html in browse
